Vim is a dated yet still beloved text editor known for its quick editing speeds, convenient (though complicated keyboard shortcuts), and simplicity.

Vim\+Together aims to replicate Vim\textquotesingle{}s original functionality and improve upon the base model by\+:
\begin{DoxyItemize}
\item simplifying complex keyboard shortcuts
\item adding live collaboration via the cloud, similar to a Google Docs
\item providing live spell check and auto complete, fueled by LLM

Here we have a list of required features for the final product.


\begin{DoxyEnumerate}
\item To be able to open, read, and write text to files of any programming language.
\item To provide convenient and useful keybinds to streamline editing files.
\item To provide a visually appealing frontend complete with language specific syntax highlighting.
\item To allow for live, asynchronous collaborative editing to a single file.
